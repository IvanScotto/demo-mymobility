Odoo Map Widget

Offering Map widget in any form view of Odoo!

Odoo Map Widget allows you to add the map widget in any of the form views in Odoo. You can simply mark the location in the map by adding its longitude along with its latitude. The module offers an ease to the customers to add map widget in core Odoo forms as well as custom.

The ease offered while working can help you to do work with the excitement and quickly. Making Odoo more convenient for you; now, you can add set the addresses easily in Odoo. All you need to set the longitude and latitude of the address.

The module offers the ease not just to the tech savies but the module also offers the ease to the non-technical people. You can add the map widget with the help of the XML code or do it with the user friendly way.

XML to add the map widget

<field name="your_field" widget="gmap_widget" lat="latitude_filed" lng="longitude_field"/>

Or you can use Map pin marker in form view.

Set the Map widget with the better User Interface
